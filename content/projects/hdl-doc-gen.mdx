HDL Doc Gen is a documentation generator specifically built for Verilog hardware description language files. It parses Verilog source code and extracts module definitions, ports, parameters, and other structural elements to generate human-readable documentation.

This tool helps hardware engineers maintain up-to-date documentation for their digital designs without the manual overhead of writing separate documentation files. Support for additional HDL languages is planned for future releases.
